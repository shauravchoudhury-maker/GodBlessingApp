# ✧ God Blessing — Daily Verse Studio

A self-contained web app (installable as a phone/tablet **PWA**) that turns daily
scripture verses into shareable **images, videos, and platform-ready captions** —
and helps you schedule and publish them.

## Features

- **Daily Studio** — independent 120-day reading plans for **Bible** and
  **Bhagavad Gita** (181 curated verses). Each day generates a verse image plus
  tailored captions + hashtags for Instagram, TikTok, Facebook, YouTube and X,
  bundled into a downloadable **Post Kit (.zip)**.
- **Translate on the page** — render the verse in 32 languages (RTL supported),
  with a **grammar / meaning check** that back-translates to English so you can
  confirm the wording before posting. Batch-export the **top 8 languages** as images.
- **Video + music** — short animated verse videos with original, browser-synthesized
  royalty-free music (exported as `.webm`).
- **Original background art** — 9 procedurally-drawn scenes (no stock photos, no
  copyright concerns).
- **Schedule & Automate** — build a multi-day posting queue and export a
  scheduler-ready **CSV + media bundle** for Buffer / Publer / Metricool / Hootsuite.
- **Native share** — on phones, one tap sends the image or video + caption straight
  to the OS share sheet (Instagram, TikTok, WhatsApp…).
- **Installable & offline** — add to your home screen; works without a connection.

## Run locally

It's a static site — any static server works. On Windows (no Node/Python needed):

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1 -Port 8123
# then open http://localhost:8123
```

## Deploy (to use on your phone)

Installing a PWA requires HTTPS. Host the folder on **GitHub Pages**, **Netlify**,
or **Vercel**, open the URL on your phone, then use the browser menu →
**"Add to Home Screen."**

## Notes on content

- Bible text uses public-domain / commonly-rendered English.
- Bhagavad Gita verses are widely-accepted English renderings (paraphrase).
- Backgrounds and music are generated by the app (original / royalty-free).
- Translation uses the free [MyMemory](https://mymemory.translated.net/) API —
  machine translation, so the built-in meaning check is there for a reason.

## Tech

Vanilla HTML/CSS/JS. No build step, no dependencies. Canvas for images/video,
Web Audio for music, MediaRecorder for `.webm`, a tiny pure-JS ZIP writer, and a
service worker for offline use.
